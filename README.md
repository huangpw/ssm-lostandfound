**项目名称：**基于SSM的校园失物招领系统（lostandfound)

**项目包名：**edu.school

**开发环境：**windows10、JDK1.8以上、MySQL5.7及以上、Tomcat7.0及以上、Maven

**前端框架：**Layui

**后台框架：**SSM+自动生成generate

**开发工具：**IDEA、Navicat

**开发作者：** 源码客栈-逍遥游

**自学作者：**huangpw

系统角色：

- 分为用户、管理员

- 分为前台和网站

前台功能有：

- 注册
- 登录
- 首页（展示失物信息）
- 发布失物
- 失物认领
- 发布帖子
- 评论帖子
- 回复帖子

后台主要有：

- 登录：管理员登录

- 用户管理：管理用户信息（修改、删除、查询）

- 失物管理：管理用户发布的失物信息

- 公告管理：管理公告信息

- 认领管理：管理认领信息（增删改查）

- 帖子管理: 管理帖子信息

- 评论管理：管理评论信息

- 回复管理：管理回复信息

- 数据统计：统计失物数量等（饼状或者柱状）

- 个人信息修改：查看和修改个人信息
